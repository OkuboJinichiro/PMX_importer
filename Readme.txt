【制作物情報】
このプログラムは学習用に制作したものです。
MMDにて使用されているPMX2.0形式のバイナリファイルをMAYAで読み込みます。

制作目標：3Dモデルデータへの理解を深める。
　　　　　バイナリファイルの扱いを学ぶ。
　　　　　MAYAにおけるデータの扱い方を学ぶ。

使用言語：Python

【使用方法】
①MAYAでプロジェクトを作成し、「script」フォルダに「Model_IMPORT」フォルダを入れる。

②start.pyをMAYAのスクリプトエディタで実行する

③PMXファイルのパスを指定して「モデル読み込み」ボタンを押す。
